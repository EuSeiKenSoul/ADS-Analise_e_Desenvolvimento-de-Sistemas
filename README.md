## 📂 Estrutura do Repositório  

Os arquivos seguem uma padronização para facilitar a organização e o entendimento:  

**Formato:** `LLNN_LLLLLLNN`  
(L = Letras | N = Números)  

**Exemplo:** `LO01_ESTSEQ01`  

- **LO01** → Lote de Exercícios 01  
- **EST** → Estrutura  
- **SEQ** → Sequencial  
- **DIC** → Decisional  
- **01** → Exercício 01
- **ALG** → Algoritmo
- **MESA** → Teste de Mesa
  

---

## 📝 Tipos de Arquivos
